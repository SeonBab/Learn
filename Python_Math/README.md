# Python-Math

이 프로젝트는 파이썬을 사용하여 matplotlib 라이브러리로 다양한 수식 그래프를 생성하고 정리하는 저장소입니다. 수학적 함수, 방정식, 데이터 시각화 등을 포함한 그래프 예제들을 포함합니다.

## 사용 라이브러리

- **matplotlib**: 그래프 시각화를 위한 주요 라이브러리
- **numpy**: 수치 계산 및 배열 연산 (matplotlib과 함께 사용)

## 설치 요구사항

- Python 3.7 이상
- pip를 통한 패키지 설치:

```bash
pip install matplotlib numpy
```

## 사용 방법

각 Python 파일을 실행하여 그래프를 생성할 수 있습니다. 예를 들어:

```bash
python example_graph.py
```

그래프는 matplotlib의 `plt.show()`를 통해 표시되거나, 파일로 저장될 수 있습니다.
