# C를 기초로 삼아 만든 언어

C++의 개발자인 비야네 스트롭스트룹(Bjarne Stroustrup)은 C의 Superset이 되길 바라는 마음으로 1970년 객체 지향적 언어인 Simula의 개념을 C언어에 추가해 "C with Classes"라는 이름으로 새로운 언어를 발표한다.
이후 증가 연산자(++)의 도입으로부터 C언어의 확장판이라는 의미를 담아 C++이라는 이름으로 1984년에 개정된다.
